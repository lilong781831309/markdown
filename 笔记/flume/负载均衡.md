#### 功能

监听端口数据,打印到logger1或logger2，logger1和logger2实现负载均衡



#### 配置

netcat-memory-avro.conf

```shell
# Name
a1.sources = r1
a1.sinks = k1 k2
a1.channels = c1

# Source
a1.sources.r1.type = netcat
a1.sources.r1.bind = localhost
a1.sources.r1.port = 6666
a1.sources.r1.selector.type = replicating

# Sink
# sink1
a1.sinks.k1.type = avro
a1.sinks.k1.hostname = hadoop10
a1.sinks.k1.port = 8888
# sink2
a1.sinks.k2.type = avro
a1.sinks.k2.hostname = hadoop10
a1.sinks.k2.port = 9999

# Channel
a1.channels.c1.type = memory
a1.channels.c1.capacity = 1000
a1.channels.c1.transactionCapacity = 100

# Bind
a1.sources.r1.channels = c1
a1.sinks.k1.channel = c1
a1.sinks.k2.channel = c1
```

#### 启动

```shell
flume-ng agent --conf $FLUME_HOME/conf --conf-file $FLUME_HOME/conf-file/load_balance/avro-memory-logger1.conf --name a1 -Dflume.root.logger=INFO,console

flume-ng agent --conf $FLUME_HOME/conf --conf-file $FLUME_HOME/conf-file/load_balance/avro-memory-logger2.conf --name a1 -Dflume.root.logger=INFO,console

flume-ng agent --conf $FLUME_HOME/conf --conf-file $FLUME_HOME/conf-file/load_balance/netcat-memory-avro.conf --name a1 -Dflume.root.logger=INFO,console

nc localhost 6666
```
