```
1.  Proxy.newProxyInstance(ClassLoader loader,Class<?>[] interfaces,InvocationHandler h) 获取代理对象
     - loader 	      用来加载新生成的代理对象 class 文件
	 - interfaces     表示在生成的代理类中为哪些接口生成代理方法
     - h			  调用代理类方法时，调用 h.invoke(this, m, new Object[] { ... });  
					
2.  class MyInvocationHandler implements InvocationHandler {
        Object target;
        public MyInvocationHandler(Object target) {
            this.target = target;
        }
		/**
		* proxy  对应  h.invoke(this, m, new Object[] { ... });   this
		* method 对应  h.invoke(this, m, new Object[] { ... });   m
		* args   对应  h.invoke(this, m, new Object[] { ... });   new Object[] { ... }
		*/
		@Override
		public Object invoke(Object proxy, Method method, Object[] args) throws Throwable {
			System.out.println("======before proxy=====");
			Object value = method.invoke(target, args);
			System.out.println("======after  proxy=====");
			return value;
		}
	}
```
